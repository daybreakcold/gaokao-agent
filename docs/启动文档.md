# 启动文档

本文档介绍如何在本地启动「高考志愿填报规划 Agent」，包括网页版和命令行版。部署到服务器请参考 [部署文档](./部署文档.md)。

---

## 1. 环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | macOS / Linux / Windows（WSL 或原生均可） |
| Python | 3.8 及以上（`python3 --version` 检查） |
| 依赖 | 仅 [`openai`](https://pypi.org/project/openai/) SDK，见 `requirements.txt` |
| 网络 | 需能访问 `api.deepseek.com`；联网搜索功能需能访问 DuckDuckGo 或 Tavily |

> 网页服务基于 Python 标准库实现，**不需要** Flask / FastAPI 等 Web 框架。

## 2. 配置 API Key

复制项目根目录的 `.env.example` 模板为 `.env`，填入真实 Key：

```bash
cp .env.example .env
```

```bash
# 必填：DeepSeek API Key，在 https://platform.deepseek.com 获取
DEEPSEEK_API_KEY=sk-你的key

# 可选：联网搜索优先使用 Tavily（更稳定），不配置则回退到免费的 DuckDuckGo
TAVILY_API_KEY=tvly-你的key
```

> ⚠️ `.env` 含密钥，已被 `.gitignore` 忽略，**请勿提交到公开仓库**；建议 `chmod 600 .env` 收紧权限。

## 3. 一键启动（网页版，推荐）

```bash
./start.sh
```

脚本会自动完成以下步骤：

1. 载入 `.env` 中的环境变量；
2. 检查 `DEEPSEEK_API_KEY` 是否存在（缺失会报错退出）；
3. 首次运行时创建虚拟环境 `.venv` 并安装依赖；
4. 自动打开浏览器；
5. 启动服务，默认地址 <http://127.0.0.1:8010>。

按 `Ctrl-C` 停止服务。

**换端口启动：**

```bash
PORT=8020 ./start.sh
```

## 4. 手动启动（网页版）

不使用 `start.sh` 时，可以手动执行：

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

export DEEPSEEK_API_KEY="sk-你的key"
python web_app.py
```

启动成功后终端会输出：

```
✅ 高考志愿填报 Agent 已启动： http://127.0.0.1:8010
```

## 5. 启动命令行版

```bash
source .venv/bin/activate
export DEEPSEEK_API_KEY="sk-你的key"

# 交互式对话
python gaokao_agent.py

# 带开场白直接提问
python gaokao_agent.py --text "江苏 2025 物化生 628分 位次9800 想去南京学计算机 服从调剂"

# 关闭联网搜索（仅用模型已有知识）
python gaokao_agent.py --no-web
```

输入 `exit` / `quit` / `退出` 结束对话。

## 6. 环境变量一览

| 变量 | 用途 | 是否必需 | 默认值 |
|------|------|----------|--------|
| `DEEPSEEK_API_KEY` | DeepSeek 鉴权 | ✅ 必需 | — |
| `DEEPSEEK_BASE_URL` | API 接口地址 | 可选 | `https://api.deepseek.com` |
| `TAVILY_API_KEY` | 联网搜索（Tavily 优先） | 可选 | 不配则用 DuckDuckGo |
| `HOST` | 网页服务监听地址 | 可选 | `127.0.0.1` |
| `PORT` | 网页服务端口 | 可选 | `8010` |

## 7. 常见问题

**❌ 未找到 DEEPSEEK_API_KEY**
`.env` 文件不存在或没写 Key。按第 2 节创建 `.env`，或先 `export DEEPSEEK_API_KEY=...` 再启动。

**端口被占用（`Address already in use`）**
换个端口：`PORT=8020 ./start.sh`；或找到占用进程：`lsof -i :8010` 后 `kill` 掉。

**浏览器没有自动打开**
手动访问 `http://127.0.0.1:8010`（端口以终端输出为准）。

**联网搜索一直失败**
检查网络能否访问 DuckDuckGo；建议在 `.env` 配置 `TAVILY_API_KEY` 提高稳定性。联网完全不可用时，Agent 会给出官方查询指引兜底，不会编造数据。

**依赖安装缓慢**
使用国内镜像：`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple`。
